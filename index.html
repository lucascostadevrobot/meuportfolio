Ecoponto

O aplicativo EcoPonto foi selecionado para participar do Evento GreenRio, sendo o único do Interior do Estado. Nesse processo seletivo partiticiparam mais de 700 Startups com projetos viáveis. Apresentamos o aplicativo EcoPonto no Estande do SEBRAE-RJ como Startup. Conseguimos realizar apresentação do produto com Pitch para investidores anjos e investidores Nacionais e Internacionais. O aplicativo EcoPonto foi desenvolvido utilizando as tecnologias Java como linguagem de Programação, XML com Android para desenvolvimento da interface, Androi Studio como IDE para o Desenvolvimento. Ao aplicativo é para o sistema operacional Androi de forma Nativa.

Biker Seeker

O aplicativo Bike Seeker ainda em desenvolvimento para o projeto de conclusão de curso (TCC) do meu curso Bacharelado em Sistemas de informações é um projeto para facilitar a vida na mobilidade urbana referente aos Motos-Boy. Depois de um estudo de campo, percebemos que existe uma certa dificuldade para os pilotos de motos que realizam transporte de passageiros. Com esse aplicativo, resolveremos esse problema na Região Norte Fluminense de forma simples, prática e organizada.
